# 国内耦合协调度模型的误区及修正（王淑佳、孔伟、任亮、治丹丹、戴彬婷）

## 一、基本信息

- 出处：自然资源学报, 2021, 36(3): 793-810。DOI: 10.31497/zrzyxb.20210319 ✅已核准
- 机构：中山大学旅游学院；河北北方学院；河北工业大学
- 类型：方法论批判与修正（高被引 CCD 方法论文）
- 本仓库状态：原文 PDF 已取，修正公式经论文表5模拟值 **6/6 数值反推验证**（误差≤0.0004）

## 二、主要内容

系统检讨截至 2018-12-31 国内 CSSCI/CSCD 期刊中 CCD 模型的使用，归纳**四类误区：书写错误、丢失系数、错用权重、模型不成立**；明确传统模型规范公式；提出两种修正方案；以京津冀 13 市生态—经济系统验证修正模型效度；并指出无论传统还是修正模型都仍存**信度问题**（指标主观性、结果波动性、无可比性）。

## 三、理论/文献引用

- 传统 CCD 谱系（物理耦合概念→区域协调发展评价）
- 对 C 值分布性质的数理分析（传统 C 非 [0,1] 均匀分布 → 效度问题源自模型本身）

## 四、公式引用（核心）

**修正方案一**：保留传统式，但按子系统**比值**重定 C 的解释边界（如二元系统比值 2:1 → C=0.9428 为拮抗/协调分界）。缺点：边界论证难、多系统复杂、"耦合还应与差值有关"——C(0.1,0.2)=C(0.4,0.8)=0.9428 但前者理应更高。

**修正方案二（式10，n 系统一般式）**：
C = √{ [1 − √( Σ_{i>j}(Uᵢ−Uⱼ)² / Σ_{m=1}^{n−1} m )] × ( Πᵢ Uᵢ/max Uᵢ )^{1/(n−1)} }

**n=2（式13，设 U₂=max）**：
**C = √{ [1 − (U₂−U₁)] × (U₁/U₂) }**
- T = Σ αᵢUᵢ, Σαᵢ=1（式11/14）；D = √(C×T)（式12）
- 性质：同时惩罚**绝对差** [1−(U₂−U₁)] 与**比值** U₁/U₂；子系统越离散 C 越低。**注意：修正 C 非尺度无关**（传统 C 只看比值）。
- 作者自认不足（§6.2）：新模型并非在 [0,1] 完全效度高，**子系统差值较小时 C 区分度不足**。

## 五、数据与验证流程

1. 文献计量：对 2018 年底前国内两库 CCD 论文逐一检查公式书写与权重使用 → 四类误区归类。
2. 数值模拟：二元/三元系统在 [0,1] 网格上模拟传统 vs 修正 C（表4/5/6），展示修正后 C 更均匀分散。
3. 实证验证：京津冀 13 市生态—经济系统，传统 vs 修正模型的 C 值（表8）、D 值（表9）对比，证实修正模型效度更好；并证实研究对象时空变化显著影响模型信度。

## 六、创新点

1. 首次系统归纳国内 CCD 四类使用误区并给出规范式。
2. 提出兼顾差值与比值的修正耦合度模型（式10/13）。
3. 坦承修正模型与信度局限——罕见的自我设界。

## 七、对本文的参考价值（直接对接 CCD 现存问题）

1. **修正C稳健性已实测**（2026-07-02，B口径2024）：九市排序基本不变（仅龙岩↔泉州、三明↔漳州两对相邻互换，方向=均衡市升）；**双核—腹地头条无恙**；corr(D修,T)=0.988（传统0.993）→ **权威修正也修不掉退化**。
2. **信息天花板证据**：Var(logD) 中 C 成分 2024 截面 12.1%→20.7%、全期 5.9%→13.0%（修正C注入绝对差新信息故可超单调天花板，但 T 仍占 80–87%）→ 支撑本文"退化非公式书写问题，而是输入语义（水平量）使然"的**退化定理**表述。
3. 写法上：主口径用传统 C（赵军2025同做法），修正 C 作稳健性表 + 引用其"作者自认差值小时区分度不足"以对冲审稿追问。
4. 其"低水平高耦合虚假性"批评 ≈ 本文"T 防低水平伪协调"的分工依据。
5. 提醒：**逐年/固定基准标准化选择会影响 C、D 可比性**（其信度之"无可比性"），与本文全网格固定基准 min-max 的选择声明相互印证。
