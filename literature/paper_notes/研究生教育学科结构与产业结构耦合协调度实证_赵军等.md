# 研究生教育学科结构与产业结构：耦合协调度的实证研究（赵军、陈佳钰、李博）

## 一、基本信息

- 出处：研究生教育研究, 2025(1): 1-10。DOI: 10.19834/j.cnki.yjsjy2011.2025.01.01
- 机构：三峡大学田家炳教育学院；湖北省教育科学研究院
- 基金：国家社科基金教育学一般项目 BIA220104
- 类型：全国时序 CCD 实证（2013–2022）

## 二、主要内容

测度我国研究生教育学科结构与三次产业结构的耦合协调：整体高度耦合，但**学科结构自 2017 年起滞后于产业结构且差距拉大**（U₁−U₂ 从 2013 年 +0.1245 转为 2021 年 −0.2072）；分层次看，农学博士、工学硕士规模扩张更迫切，人文社科相对规模宜稳、博士层次需扩。政策节奏：短期调培养规模、中期补学科建设短板、长期完善招生调节机制。

## 三、理论引用

- 系统论（学科结构与产业结构为两大子系统，CCD 适合系统间关联分析）
- 学科—产业匹配划分：农学→一产、理工→二产、人文社科→三产（引杨林、雷云等 [15][16]）
- 人力资本作用滞后性（据此设计教育侧滞后一年）
- 方法学引用：**引了王淑佳等(2021)误区及修正[文献10-12 区间]，但仍采用传统 C 公式**

## 四、公式引用

- C = 2√(U₁U₂)/(U₁+U₂)（传统式）
- T = α·U₁+β·U₂，α=β=0.5；D = √(C×T)
- 分级：耦合 5 级（0.2 等距：极低/轻度/中度/良性/高度）；协调 8 类（0.2/0.4/0.5/0.6/0.7/0.8 分界，从极度失调衰退到优质协调发展）

## 五、数据收集与处理流程

1. **U₁（学科结构）**：各学科毕业生数 / 当年度全体毕业生数（**占比**）；分"全体/硕士/博士"三套层次口径。
2. **U₂（产业结构）**：三次产业产值 / GDP（占比）。
3. **滞后设计（要点）**：考虑毕业研究生人力资本作用滞后，**教育侧取 t−1 年（2012–2021），产业侧取 t 年（2013–2022）**。
4. 数据源：教育部网站 + 历年《中国统计年鉴》。
5. 剔除规则：军事学、新设交叉学科因特殊性**不纳入**。
6. 标准化：极值法；赋权：**熵值法**（12 个学科毕业生比重 + 3 个产业比重，权重表按全体/硕士/博士分列）。

## 六、检验过程

无回归；属测度—趋势型：U₁、U₂、T 时序表（2013–2022）+ U₁−U₂ 方向判定（超前/滞后）+ 硕/博分层趋势图 + 农学↔一产、理工↔二产等细分领域对照。2020 年 U₂ 大幅回落归因新冠。

## 七、创新点

1. 把学科—产业占比 CCD 细分到**学位层次维度**（硕/博），超出以往科类维度。
2. 明确的教育侧 t−1 滞后设计。
3. "整体高耦合 + 方向性滞后"的二元诊断（不满足于 D 排名）。

## 八、对本文的参考价值（含 CCD 现存问题）

1. **占比范式第二证据**：与商润泽合成"结构占比 CCD"谱系（国别→全国研究生），坐实该范式活跃；本文（省域×AI×功能链）填的空更清晰。
2. **传统C+引而不用修正式的先例**：直接支持本文"主口径传统 C、王淑佳修正式仅作稳健性"的处理。
3. **U₁−U₂ 方向分析 = 本文 L=E−I 的又一先例**：其"2017 起学科滞后"与商润泽"中等收入国教育滞后"互证——**方向量（领先/滞后）才是这类研究的判别主力，D 只作背景**，再次支撑本文"报 M/L、D 降级"。
4. **t−1 滞后设计 vs 本文同期构念**：可在方法节作为对照选项讨论——本文构念定为"同期状态"（水平匹配），发育/滞后交给轨迹分析；若审稿人问滞后，引赵军做法说明"滞后设计适用于毕业生存量口径；本文 E1 为招生承诺口径，同期状态更自洽"。
5. 剔除特殊学科（军事学/交叉学科）的声明写法可借鉴（对应本文剔除高职专科/征求志愿的口径声明）。
