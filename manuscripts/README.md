## 重要说明

1. **参考文献无虚构**：正文与文末引用均为高置信度真实文献；凡不确定准确著录信息者，已显式标注【待核验】或【需作者补充】，请投稿前逐条核对卷期页码。障碍度模型的方法学引用须作者补入所采用版本（本稿不为此编造引用）。
2. **结果为假设性示例**：论文 B 两版的回归表数值均为示意，已在表题标注"（假设性示例）"，用于展示分析结构与读法，待真实数据管道运行后替换。论文 A 第5—6章给出结果的呈现结构，数值待全样本管道填入。
3. **三篇共享内核**：同一套数据、口径（C/B/X、人均化、当期 E）、功能链（供给规模/供给质量/产业承接/产业转化 + 传导桥）、CCD 与障碍度引擎。论文 A 提供测量合法性；论文 B 两版差别在识别深度。
4. **红线**：金融（F）与技术暴露变量仅进回归右端，绝不进入 CCD 构造。
