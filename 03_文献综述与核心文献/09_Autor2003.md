# Autor, D.H., Levy, F. & Murnane, R.J. (2003)

## 基本信息
- **作者**：David H. Autor, Frank Levy, Richard J. Murnane
- **年份**：2003
- **发表**：*Quarterly Journal of Economics*, 118(4), 1279-1333
- **核心问题**：计算机化如何改变不同职业的技能需求？

## 核心论点

### 1. 技能任务框架（Task-Based Framework）
- **职业** = 不同"任务"的组合
- 计算机擅长：**routine cognitive/physical tasks**
- 人类擅长：**non-routine cognitive/interpersonal tasks**

### 2. 计算机化对职业的替代效应
| 任务类型 | 计算机化可能性 |
|---------|--------------|
| Routine cognitive | 高 |
| Routine manual | 高 |
| Non-routine cognitive (analytical) | 中 |
| Non-routine cognitive (interactive) | 低 |
| Non-routine manual | 低 |

### 3. 结构性变化预测
- 低技能+Routine → 高失业风险
- 高技能+Non-routine → 需求增加
- 教育结构需跟上技能需求变化

## 对本研究的启发
- ✅ **AI时代的技能需求变化** → 解释为什么AI学科结构调整必须与产业同步
- ✅ **任务框架** → 可用于分析AI岗位的技能需求特征
- ⚠️ 该研究聚焦美国，中国AI产业可能有不同的技能需求结构
