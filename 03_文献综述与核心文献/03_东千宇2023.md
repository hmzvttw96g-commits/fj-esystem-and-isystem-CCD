# 东千宇、钟凯毅 等 (2023)

## 基本信息
- **作者**：东千宇、钟凯毅、廖一佳、熊润丽、王峰波、庞敏
- **年份**：2023
- **发表**：*Resources Policy* (Elsevier, JCR Q1)
- **DOI**：https://doi.org/10.1016/j.resourpol.2023.103308
- **核心问题**：中国资源型省份能源-经济-环境(3E)系统的耦合协调度如何？

## 核心论点

### 1. 三系统耦合模型（3E系统）
- 能源子系统 + 经济子系统 + 环境子系统
- 三个子系统相互影响、协同发展

### 2. 主要发现
- 2011-2020年间，中国资源型地区3E系统耦合协调度从0.541提升至0.643
- 东部 > 西部 > 中部
- 广东省耦合协调度最高，青海最低

### 3. 指标体系（可直接参考）

#### 能源子系统（f₁）
| 一级指标 | 二级指标 | 权重 |
|---------|---------|------|
| 能源结构 | 天然气消费比重、油耗百分比 | 0.058-0.099 |
| 能源安全 | 自给率、研发成本 | 0.037-0.120 |
| 能效 | 人均能耗、单位GDP能耗 | 0.032-0.074 |

#### 经济子系统（f₂）
| 一级指标 | 二级指标 | 权重 |
|---------|---------|------|
| 经济规模 | 人均GDP、人均可支配收入 | 0.086-0.141 |
| 经济结构 | 一产/二产/三产增加值 | 0.091-0.128 |
| 经济效率 | 劳动生产率、固定资产投资 | 0.048-0.108 |

#### 环境子系统（f₃）
| 一级指标 | 二级指标 | 权重 |
|---------|---------|------|
| 环境质量 | 森林覆盖率、PM2.5 | 0.046-0.125 |
| 污染治理 | 废水处理率、固废处置率 | 0.049-0.146 |

### 4. 三维耦合协调度公式
```
C = (f₁ × f₂ × f₃) / [(f₁ + f₂ + f₃)³]
T = (f₁ + f₂ + f₃) / 3
D = √(C × T)
```

### 5. 分级标准（10级，更细化）
| D值 | 等级 | 类型 |
|-----|------|------|
| 0.9-1.0 | X | 高质量耦合协调 |
| 0.8-0.9 | IX | 良好耦合协调 |
| 0.7-0.8 | VIII | 中级耦合协调 |
| 0.6-0.7 | VII | 初级耦合协调 |
| 0.5-0.6 | VI | 弱耦合协调 |
| 0.4-0.5 | V | 濒临失调 |
| 0.3-0.4 | IV | 轻度失调 |
| 0.2-0.3 | III | 中度失调 |
| 0.1-0.2 | II | 严重失调 |
| 0-0.1 | I | 极度失调 |
