# 论文A 双路径产出规划：课题版 × 顶刊版

> 一套结果，两个门面。课题版（福建省 AI 教育产业政策分析）与顶刊版（测量诊断方法论文）
> **共享同一台引擎**，差别只在"把什么放台前、什么放幕后"。本规划定义共享内核 + 两条产出路径 + 同源结果的呈现对照 + 各自独需的额外工作 + 推进顺序。
> 依赖：docs/口径冻结文档_v1.0_草案.md、功能链设计（4环节+传导桥）、docs/论文A框架_测量与诊断篇_CSSCI.md。

## 一、共享内核（建一次，两版复用）

| 模块 | 内容 | 状态 |
|---|---|---|
| 样本 | 福建 9 设区市 × 2019–2024 | 锁定（两版都走9市；扩样只是论文B的事） |
| 口径 | C/B/X 三套；人均化；**当期 E（不滞后，主口径）** | 已决（滞后/分类约束所致） |
| E 功能链 | 供给规模（本科招生）+ 供给质量（层次·一流专业） → E_index | 待formalize |
| I 功能链 | 产业承接（企业存量+岗位）+ 产业转化（企业专利） → I_index | 待formalize |
| 传导桥 | 留存↔流失，F 作用点，**不入 CCD** | 概念命名（论文B填F） |
| 测度 | CCD(E,I) + **障碍度 4 环节分解**（诊断引擎） | 核心 |
| 派生读数 | 四类城市（类型）、三类路径（轨迹）——均由障碍度/D 导出，单一引擎 | 待formalize |
| 边界与复核 | E/I/F/G 边界判定；双人编码 κ；口径冻结文档 | 部分就绪 |

**两版唯一共同的纪律**：障碍度是唯一引擎，类型与路径都是它的读数，口径全程冻结一致。

## 二、路径一 · 课题版（政策分析报告）

- **定位**：福建省 AI 教育—产业协调发展的诊断与**差异化政策**
- **读者**：导师课题组、省/市政策部门
- **门面**：四类城市 × 障碍度政策矩阵（哪些城市 × 卡哪一环 → 具体政策）

**结构**
1. 研究背景与政策需求（福建 AI 教育产业现状）
2. 测度框架（简述 E/I 功能链、CCD、障碍度——够用即可，**不做文献批判**）
3. 福建 9 市 E-I 协调现状（CCD 水平、时空格局、**四类城市分布**）
4. 障碍度诊断（各市卡在哪一环——政策精度来源）
5. **差异化政策矩阵 ★核心交付**（城市类型 × 障碍环节 → 具体政策）
6. 动态视角（路径类型：哪些市改善/停滞 → 政策优先级）
7. 结论与政策建议清单

**特点**：篇幅给政策、城市画像、可操作建议。**不需要**文献缺口批判、安慰剂、F 端、识别策略。
**形态**：课题结题报告，可再拆出政策简报/内参。

## 三、路径二 · 顶刊版（测量诊断方法论文）

- **定位**：系统边界、功能分解与教育—产业耦合协调测度
- **读者**：CSSCI/顶刊审稿人（《教育与经济》《管理世界》路线）
- **门面**：障碍度引擎作为方法贡献 + 朴素vs清洗对比

**结构**
1. 引言（从测量缺陷切入，不从福建/政策切入）
2. 文献与方法批判（边界混杂/黑箱总分/事后调参 三缺陷 + **§2.4 魏文(2026)划界**）
3. 测度体系构建（E/I/F/G 边界 + 功能链 + C/B/X + **口径冻结** ★方法贡献）
4. 数据（双人编码 κ、采集日志、抽取完备性）
5. 测量结果（CCD 水平 + **朴素CCD vs 清洗CCD 对比 ★创新点1实证落点**）
6. 障碍度功能诊断（4 环节分解 ★创新点2）
7. 稳健性（C/B/X、当期vs滞后、**非AI专业群安慰剂**）
8. 结论（方法论建议 + **简短**政策含义）
- 附录：口径冻结文档（A域核心）、pilot 失效模式、κ 检验、run_id 索引

**特点**：篇幅给方法、严谨、新颖性。典型城市/政策**压缩成一小节**；四类城市**降为一张派生图**。
**形态**：期刊论文。

## 四、同源结果，两种呈现（crosswalk）

| 同一份结果 | 课题版 | 顶刊版 |
|---|---|---|
| 障碍度 4 环节 | 政策精度来源（§4） | **方法贡献（§6，创新点2）** |
| 四类城市 | **政策门面，核心（§3、§5）** | 派生读数，一张图 |
| 三类路径 | 政策优先级（§6） | 一句话/一图 |
| 朴素vs清洗对比 | 不强调 | **创新点1实证落点（§5）★** |
| 口径冻结 | 附录提一下 | **可信度核心（创新点4）** |
| 魏文划界 | 不需要 | **必须（§2.4）** |
| 差异化政策矩阵 | **核心交付 ★** | 压缩成"政策含义"一节 |
| F 端 / 虹吸 | 不出现（留论文B） | 不出现（留论文B） |

一句话：**课题版以"类型×障碍度政策矩阵"领衔，顶刊版以"障碍度引擎+边界清洗"领衔——同一套结果换门面。**

## 五、各版独需的额外工作（已知工作量差）

**课题版独需**：政策矩阵细化、城市画像撰写、可读性（去术语）。
**顶刊版独需**：①文献缺陷批判 ②魏文划界 ③**朴素vs清洗对比的实证**（边界污染偏误量化——这是创新点1，必须实做）④非AI安慰剂 ⑤双人编码 κ 全量 ⑥口径冻结文档全文附录。

> 关键判断：顶刊版的额外工作里，**只有"朴素vs清洗对比"和"双人编码κ"是新增分析**，其余是写作/附录。所以顶刊版不是重做，是"在课题版结果上加方法包装 + 两项新增分析"。

## 六、推进顺序（最省力路径）

```
共享内核（数据 + 口径冻结 + CCD + 障碍度 + 四类城市/路径）
   │
   ├─→ 课题版先出 ←← Fujian-9 就绪、是课题义务、当期口径不卡、不依赖魏文/扩样
   │      产物：结题报告 + 政策矩阵
   │
   └─→ 顶刊版（复用全部结果）
          + 补两项新增分析（朴素vs清洗、双人κ）
          + 加方法包装（文献批判、魏文§2.4、口径冻结附录、安慰剂）
          产物：CSSCI/顶刊投稿
```

**理由**：课题版是义务、且数据就绪可立即推进；顶刊版站在课题版的结果之上，只补两项分析+换门面，边际成本最低。**先课题、后顶刊，绝不并行重做。**
