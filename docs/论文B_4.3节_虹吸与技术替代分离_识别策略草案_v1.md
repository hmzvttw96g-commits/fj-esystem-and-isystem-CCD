## 4.3.4 一个必须排除的替代解释：技术替代而非金融虹吸

### （1）识别威胁的陈述

本文 H2 主张：金融部门对 AI 基础专业群人才的吸纳（虹吸）越强，教育供给向产业端的有效传导越弱，E-I 耦合协调度（CCD）越低或改善越慢。虹吸强度的主代理为年鉴类金融业相对工资溢价与金融业城镇单位就业占比（与岗位数据完全独立，对应 Philippon–Reshef 的工资—配置机制）。

然而，一条与之**同向**、且作用对象**高度重合**的外生通道会污染对 β₂ 的识别。魏立佳等（2026）以"任务—职业—专业"三级传导模型测度了 816 个本科专业的"远期暴露率"，并发现：**高暴露专业普遍具有结构化课程体系与标准化就业路径**——这恰好覆盖计算机、软件、数据科学等本文 B 口径 AI 基础专业群的核心成员；且**暴露率越高，专业对口岗位的招聘份额下降越明显**。

由此产生的混杂可表述为：金融集聚城市观测到的 E-I 耦合改善偏慢、对口岗位份额偏低，**究竟是金融部门把这批人才虹吸走了（本文机制），还是 LLM 使这批专业对应的产业任务本身被自动化、从而压低了 I 端对其的有效需求（技术替代机制）？** 两条通道都压低同一个 CCD，且其共同作用对象——数理能力强的 AI/计算机/量化类毕业生——高度重叠。若遗漏技术替代通道，β₂ 将吸收本应归于技术替代的那部分方差，金融虹吸效应被高估。

本节通过**三条相互独立的识别杠杆**分离两条通道，并以稳健性主表（表 4.X）集中呈现。

> ⛔ 纪律红线：技术暴露变量自始至终只进入回归右端，绝不进入 E_index / I_index / CCD 的构造。这与 F 端不入 CCD 的处理一致，守住机械内生防火墙。

### （2）分离的逻辑基础：两条通道的变异来源正交

分离之所以可能，在于两条通道的**变异来源在维度上正交**：

- **技术替代通道**的强度主要随 **专业** 与 **时期** 变化：给定一个专业（如计算机），其远期暴露率是由该专业的任务结构决定的全国性属性，福州的计算机专业与厦门的计算机专业暴露率近似相等；其"咬合"程度则随 LLM 能力跃迁的时点（2022 年底 ChatGPT 大规模可用）而变化。因此技术替代本质是一个 **专业 × 时期** 属性，**在城市间几乎不变**。
- **金融虹吸通道**的强度主要随 **城市 × 时期** 变化：厦门的金融业工资溢价远高于泉州，这是城市层面的劳动力需求属性，**在专业间不以暴露率的方式变化**。

正是这种"一个主要在专业维度变、一个主要在城市维度变"的结构，使得在城市面板中通过以下三条杠杆把二者剥离开来。

### （3）三条识别杠杆

**杠杆一（最干净）：时序分离 —— LLM 暴露在 2022 年前近乎不咬合。**

金融科技扩张对 AI/计算机/量化人才的虹吸在样本前期（2015—2021）即已强烈，**早于** LLM 技术替代成为现实约束（生成式大模型在 2022 年底才大规模可用）。因此：

> 若在 **t ≤ 2021 子样本**（此时技术替代通道近乎静默）上估计 H2，得到的显著负向 Siphon 系数**不可能**来自 LLM 替代，只能来自劳动力再配置的虹吸通道。再证全样本系数与该子样本系数方向一致、量级相近，则技术替代即便存在，也不构成 β₂ 的主要驱动。

这是一个利用本文面板时间结构（2015—2024）的天然安慰剂，识别力最强，且不依赖暴露率数据的基数精度。
*代价：子样本观测量收缩（福建 9 市 × 约 7 年，扣滞后后有效样本偏小）。这再次指向 WP0 预检——若扩展至闽浙粤地级市，聚类升至 30+，时序子样本的推断力同步改善。⚑*

**杠杆二（直接控制）：构造城市层"暴露加权供给构成"作为控制变量。**

利用魏立佳等（2026）的专业层远期暴露率，构造本文城市—年份层面的技术暴露控制项：

```
Expo_it = Σ_{m∈B口径} w_{i,m,t-4} · Expo_m
```

其中 `Expo_m` 为专业 m 的全国远期暴露率，`w_{i,m,t-4}` 为城市 i 在 t-4 年（与 E 端培养周期滞后一致）AI 基础专业群招生计划中专业 m 的份额。`Expo_it` 度量"该城市教育供给在多大程度上集中于高 LLM 暴露专业"。再以 `Post_t = 1{t ≥ 2023}` 标识技术替代开始咬合的时期，技术替代通道近似为 `Expo_it × Post_t`。在基准回归中纳入 `Expo_it` 与 `Expo_it × Post_t`；若 Siphon 系数 β₂ 在控制后依然稳健，则金融虹吸与"本市供给恰好集中在高暴露专业"两件事被分离。

> ◆ 数据可得性：`Expo_it` 完全由本文已有的 E 端逐专业招生计划份额 + 魏文专业层暴露率合成，**无需新增一手采集**，可得性高。
> ◎ 稳健性处理：鉴于魏文暴露率存在"模型自评能力"的循环偏差，正文不依赖其基数精度——用暴露率**三分位序数**（高/中/低暴露组）替代连续值重估，证明结论对暴露率的测量误差不敏感。

**杠杆三（最强、视数据可得性）：背离预测的"赛马"检验。**

两条通道对"效应应集中在哪类专业"给出**可分辨的背离预测**：

- 金融虹吸预测：负效应应集中在**金融部门实际招聘强度高**的专业（CS、量化、数理统计），与这些专业的 LLM 暴露度本身无关；
- 技术替代预测：负效应应集中在 **LLM 暴露度高**的专业，与金融是否招聘无关。

二者在如下专业上分叉：高金融招聘—低暴露专业（如偏判断/沟通的金融工程相关）、低金融招聘—高暴露专业（如偏程式化的数据处理类）。据此构造两个城市层构成指数 `FinHire_it = Σ w·FinHire_m`（专业 m 向金融部门输送强度）与 `Expo_it`，做"赛马"回归：

```
CCD_it = α + β₁·FinDev_{i,t-1} + β₂·(Siphon_{i,t-1} × FinHire_it)
         + β₃·(Expo_it × Post_t) + γ·X_it + μ_i + λ_t + ε_it
```

若 β₂ 显著为负而 β₃ 不显著/弱，则金融虹吸主导；反之技术替代主导。
*◆ 数据可得性：`FinHire_m`（专业→金融部门输送强度）依赖高校就业质量报告的去向结构，口径不一、可得性为中。故杠杆三定位为"理想识别"，可得则单列一节作为最强证据，不可得则以杠杆一、二承担主识别，框架不依赖它。*
*⚠️ 共线提示：`FinHire_m` 与 `Expo_m` 在专业层正相关（CS 既高金融招聘又高暴露），但城市间专业构成 `w_{i,m}` 不同提供了识别变异；且杠杆一的时序分离完全绕开此共线，三杠杆互为后备。*

### （4）计量设定

在第4.1节基准模型上扩展，分离回归的完整设定为：

```
CCD_it = α + β₁·FinDev_{i,t-1} + β₂·Siphon_{i,t-1}
         + β₃·Expo_it + β₄·(Expo_it × Post_t)
         + γ·X_it + μ_i + λ_t + ε_it
```

- 被解释变量 CCD ∈ [0,1]：fractional logit 主报告，线性 FE 对照（同第4.1节）。
- 解释变量统一滞后一期（缓解反向因果）；`Expo_it` 的供给权重用 t-4 滞后（与 E 端培养周期一致）。
- 推断：wild cluster bootstrap（Webb 权重，应对少聚类）+ Driscoll–Kraay 双报告（同第4.2节）。⚑ 9 聚类下尤须 wild bootstrap；扩样后缓解。
- `Post_t = 1{t ≥ 2023}`（生成式大模型大规模可用的次年起）；时点选择做 2022/2023 双版本敏感性。

### （6）与其他模块的接口与诚实局限

- **与第7.3节安慰剂的衔接**：现有"非 AI 专业群伪 E_index → F 系数应消失"的二元安慰剂，可升级为**剂量—反应版**：沿暴露率（或 FinHire）梯度，金融虹吸效应呈理论预期的单调变化。二元安慰剂证"非机械产物"，剂量版证"机制定向正确"。
- **与论文A 的接口**：`Expo_it` 的供给权重 `w_{i,m,t-4}` 取自论文A 冻结口径下的逐专业招生计划，**不重做测量**（⚑ 测量合法性外包给论文A）；暴露率 `Expo_m` 为外部文献产出，明确标注来源与其循环偏差局限。
- **诚实局限（须在正文写明）**：① 魏文暴露率为全国值，未刻画同一专业的地区培养质量差异，本文以城市 FE 部分吸收；② `Post_t` 的二元时点是对技术咬合的粗近似，连续化（如按 LLM 能力指数）留待数据可得后扩展；③ 若 `Expo_it` 与 `Siphon_it` 城市层相关过高（VIF 须报告），则以杠杆一（时序分离）为主识别、杠杆二为辅。
