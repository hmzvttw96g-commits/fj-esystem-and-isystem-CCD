## 〇、总原则（先记这一条）

**方法由"问题类型"决定，不是由"想不想高级"决定：**
- 问"协调怎么演化" → **描述性动态**（趋势/收敛/迁移），无处理，不碰 DID。
- 问"连续变量 X 怎么影响 CCD" → **面板回归**（X 平滑变化，非二元处理）。
- 问"某离散冲击的因果效应" → **DID/事件研究**（必须有 treatment：某些单元某年被干预、其余没有）。

> ⛔ DID 的"差分"是**处理前后**的差，不是**时间趋势**的差。没有离散处理，就没有 DID 要估计的对象。

## 二、动态诊断三件套（描述性，课题版核心、顶刊版动态稳健性）

**① 水平 × 趋势，并看不偏废**：CCD 水平答"现在站在哪"，ΔCCD 答"在改善还是恶化"。丢掉水平答不了现状，丢掉趋势答不了成效。

**② 障碍度瓶颈迁移 ★独门武器**：追踪每个城市障碍度的**主环节随时间怎么迁移**（如某市 2019 卡「供给规模」→ 扩招后 2024 迁移到「产业承接」）。静态 CCD 给不了——它把功能链与动态焊在一起，是政策金矿（"约束从学校端移到企业端"），顶刊与课题都吃得开。

**③ 趋同分析**：
- **σ-收敛**（首选，9 市稳健）：逐年算 9 市 CCD 离散度（CV/SD），下降=趋同；正式检验=离散度对时间趋势回归。
- **β-收敛**（谨慎）：CCD 增长对初始水平回归，负号=落后市追赶；9 个点偏弱，仅作描述性、带caveat。
- **Markov 转移矩阵**（可选）：CCD 等级间转移概率；9 市格子稀疏，锦上添花不当主菜。

## 三、因果识别（论文 B 顶刊版）

**主检验**：分数 logit（CCD∈[0,1]）+ 线性 FE 对照；解释变量滞后一期；wild cluster bootstrap（Webb 权重）+ Driscoll–Kraay 双报告。

**为什么虹吸主检验不能 DID**：金融发展/工资溢价在城市间是平滑连续变化，不是二元处理。连续处理 DID 本质退化为面板 FE，所以面板回归就是对的方法。

**DID/DDD 的三个合法落点**：
1. **LLM 冲击（2022末）**：`Expo×Post` 已是 DID 味事件研究（Post=2022后，处理强度=专业暴露率）。
2. **政策/金改试验区**：若某些市某年获批、有差异化暴露 → 事件研究/DID，最强因果模块（§7.5可选）。这是金融侧唯一可 DID 的入口（批复离散，连续金融深化不是）。
3. **三重 DID（识别升级）**：(城市金融暴露 × 冲击后 × **AI组 vs 非AI组**)。第三重差分净掉"城市×时间"和"城市×专业组"混杂——把 §7.3 剂量安慰剂升级成 DDD，更锐利。

> ⚠️ 关键澄清：DID/DDD 识别的是 **LLM/政策冲击**的效应，**不是直接估虹吸**（虹吸连续→面板）。DID 的作用是**净化/坐实虹吸估计**（证明耦合恶化不是技术替代造的），不是替代虹吸主检验。

## 四、硬约束与避坑

1. **N=9 的天花板**：① system GMM 不可用（要大 N）；② 带滞后因变量的 FE 有 Nickell 偏误（T=6 太短）；③ DID 少处理/控制组 → 少聚类推断脆弱、平行趋势仅 3 个冲击前年份太薄。**→ 动态因果与 DID/DDD 的功率，9 市买不起，靠 WP0 扩闽浙粤补。**
2. **CCD 有界 [0,1]**：ΔCCD 有天花板效应（D=0.9 想涨涨不动）。**强推断用 logit 变换后的 CCD 再差分；描述性趋势用裸 ΔCCD 无妨。**
3. **三重 DID 需组层结果**：被解释变量得是专业组层面伪 CCD（§7.3 已构造 `CCD^g`），城市层单一 CCD 做不了 DDD；组层 CCD 的构造与解释比城市层复杂，需想清楚。
4. **防火墙不变**：暴露率、F 变量只进回归右端，绝不进 CCD 构造。

## 五、双路径分配

| 方法 | 课题版 | 顶刊版 |
|---|---|---|
| 瓶颈迁移 | ★政策核心 | 创新点动态延伸 |
| σ-收敛 | ★全省均衡/极化 | 描述性动态 |
| E−I 缺口轨迹 | 政策（错配修复否） | 稳健性 |
| 面板主检验（虹吸） | 不需要（留B） | ★核心 |
| ΔCCD 回归 | 不需要 | F 对改善速度 |
| Expo×Post / DID | 不需要 | 通道分离 |
| 三重 DID | 不需要 | 识别升级（需扩样） |
| system GMM | 不碰 | 仅参考，9 市不可主张 |

## 六、依赖与待定

- 三重 DID 与 ΔCCD 因果回归**依赖 WP0 扩样结论**（9 市功率不足）；课题版的描述性动态**不依赖扩样**，9 市即可做。
- 是否有可用的离散政策/金改冲击（决定 §7.5 与 DDD 的"Post"维度）→ 待查省级 AI 教育/数字经济/金融改革试验区批复的城市差异化暴露。
- 瓶颈迁移、σ-收敛的具体口径（障碍度主环节判定规则、离散度指标选 CV 还是 SD）→ formalize 功能链规范时一并定。
