## 一、研究问题（三问，全部是测量与诊断层面）

1. 现有 CCD 文献在测量教育—产业耦合时存在哪些系统性偏误？（边界混杂、黑箱总分、事后调参）
2. 如何构建一套边界清晰、功能可分解、口径事前冻结的 E-I 耦合测度体系？
3. 应用该体系：福建省各城市 E-I 耦合处于什么水平？失调由哪个功能环节贡献？

**全文表述纪律：** 一切结论以"测量结果"与"诊断性证据"措辞呈现，不做因果归因，不讨论"为什么"。"为什么"留给论文B。

## 二、章节框架

### 第1章 引言
- 现实背景：AI 技术扩散下的人才供需错配；地方政府对"教育—产业协调"的政策需求与测量工具的粗糙之间的矛盾
- 切入点：CCD 方法被大量用于教育—产业研究，但三大测量缺陷从未被系统处理
- 贡献预告（见创新点清单）

### 第2章 文献与方法批判
2.1 CCD 方法源流及其在教育—产业领域的应用现状
2.2 三大缺陷批判（本文的靶子）：
- **缺陷一：系统边界混杂** —— 高校科研岗、政府信息化岗、金融机构 IT 岗被无差别计入"产业需求"，导致 I 端系统性高估且偏误方向城市间不一致
- **缺陷二：黑箱总分** —— CCD 只输出一个数，失调时无法定位环节，政策含义空泛
- **缺陷三：权重与口径的事后弹性** —— 文献中普遍存在按结果反推口径的嫌疑，缺乏事前承诺机制
2.3 教育—劳动力市场管道时滞文献（支撑 E 端滞后处理）

### 第3章 测度体系构建 ★核心章节
3.1 **E/I/F/G 四端系统边界判定框架** ★创新点1
- 判定总则：高校岗→E_extension；企业产业岗→I_job；金融机构岗→F（排除，仅一行带过 ⛔）；政府事业单位岗→G_public_sector
- 研究员岗位三步判定法：employer_sector → job_function → system_mapping
- 证据分层规则：岗位证据（计数）vs 项目证据/企业名单证据（不计数，仅佐证）
3.2 E_index 构成
- **总体规则（2026-06-12 裁定）**：全部本科层次高校（公办、民办、独立学院、职业本科），高职专科剔除——总体定义与口径维度正交，不设公办/民办对比序列；办学性质异质性三层处理：①"民办计划占比 × C/B/X 口径"梯度表（民办供给集中于功能链基础层、向 AI 核心层份额递减的经验证据）②学科评估质量权重稳健性 ③研究生层次高端供给的测量边界声明
- B 基本口径专业群清单（AI、计算机、软件、网工、物联网、数据科学、智能科学、网安、信安等）
- **人均化处理**：招生计划原始数除以城市常住人口后再标准化，剥离城市规模混杂（未人均化版本作对照报告，量化规模混杂幅度）
- **滞后处理**：主口径用 t-4 年招生计划对齐培养周期；当年值做稳健性 ◆
- **抽取完备性检查协议**：每个城市—年份对照教育部高校名单枚举该市全部本科院校，零命中学校须人工确认为真零方可进面板（pilot 证明覆盖不稳定是 E 端最大测量风险：厦门 2019—2020 仅命中 1—2 所高校）◎
- **代理边界的诚实讨论**：正文承认"招生计划≠录取≠毕业≠留本市"的迁移偏误，且该偏误与城市吸引力相关、属系统性；留闽率/留本市率调整与永续盘存累积供给（回应流量—存量混搭质疑）做稳健性
- 跨校区高校归属核验协议（华侨大学方案）★创新点5
3.3 I_index 构成（功能模块化）★创新点2
- **主口径 = I_firm 企业存量 + I_patent 专利申请双模块**（人均化后合成，模块权重事前冻结为等权）
- **I_job 降级（已于 pilot 阶段触发）**：官方源岗位数据稀疏与边界清洗规则共振，在民营经济城市机械产出 I_job=0（泉州 2019 候选 0 条、2020 仅 2 条且分流后归零）；I_job 降级为 2021—2024 子样本稳健性模块，该决定先于全样本 CCD 做出并写入冻结文档 ◎
- C/B/X 三套固定口径（保守/基本/扩大），统一适用所有城市，禁止城市动态调权
- 专利按申请人地址归城市、剔除高校申请人；校企联合申请的归属规则一并写入冻结文档
3.4 口径冻结声明 ◎ ★创新点4
- 全部规则于三城市预检验阶段（pilot）锁定；全样本分析零调整
- 冻结文档全文作为在线附录
3.5 CCD 计算与功能分解方法
- 标准 CCD 公式 + 失调贡献度按功能模块（E供给/I企业/I创新；I岗位仅 2021—2024 子样本）的分解恒等式
- **零值退化处理**：任一指数为 0 时 D 机械退化为 0，属公式性质而非经济事实；零值观测标记"不可计算"单独报告，不参与协调等级与路径分类 ◎
- **标准化冻结**：全样本固定基准 min-max；秩标准化做稳健性；熵权法因随样本重算权重违反口径冻结，仅作披露性稳健性 ◎

### 第5章 测量结果
5.1 全样本 CCD 水平与时序动态（分城市；人均化主口径，未人均化对照表量化规模混杂；零值观测单独列报，不参与路径分类）
5.2 三口径稳健性（C/B/X 对照）
5.3 **朴素 CCD vs 边界清洗后 CCD 的对比** ★创新点1的实证落点
- 量化边界污染的偏误幅度与方向
- 重点展示：偏误在不同城市方向不一致（金融集聚城市 I 端高估更重），故污染不仅抬高水平、还会扭曲城市间排序——这是"为什么必须做边界清洗"的最强证据

### 第6章 失调的功能性诊断
6.1 分解结果：失调贡献度按功能模块、按城市类型、按时期
6.2 典型城市画像（如：泉州型——I 端岗位证据维度滞后；福厦型——E 端供给增速快于 I 端吸纳）

### 第7章 结论
7.1 主要发现（测量层面）
7.2 对 CCD 应用研究的方法论建议

7.4 局限与展望——结尾留一个钩子："耦合失调的驱动因素，特别是其他系统对 E-I 耦合的外部影响，留待后续研究"（为论文B埋引用接口，但不点破 F 端 ⛔）

## 五、投稿策略

- **目标档位**：《教育与经济》《华东经济管理》《教育发展研究》一档 CSSCI。不投更弱的刊——本文承担论文B的方法论背书，发得太水背书贬值。
- **审稿人预期质疑与回应**：
  1. "只是测量，没有因果" → 引言即明确定位为测度方法论文，对标指数构建类文献（如数字普惠金融指数测度论文的先例）
  2. "为何不直接用供需比" → CCD 同时刻画共同发展水平与协调度两维；且本文贡献恰在修正 CCD 而非捍卫 CCD
  3. "人工判定可复现吗" → 附录C/D：双人编码 κ 值 + 全量判定留痕
  4. "城市间 CCD 差异是不是城市规模差异" → 人均化主口径 + 未人均化对照表，把规模混杂的幅度直接亮给审稿人
  5. "招生计划数能代理本地人才供给吗" → 正文诚实承认"计划≠录取≠毕业≠留本市"三层断裂及其系统性方向；留存率调整与累积供给存量做稳健性
- **与论文B的引用接口**：本文投出后即生成 working paper 编号，论文B引用该版本；本文录用后 B 改引正式版。
