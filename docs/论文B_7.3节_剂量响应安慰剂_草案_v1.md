## 7.3 剂量—反应安慰剂：效应是否定向跟随金融招聘强度

### （1）从二元到剂量：为什么要升级

二元安慰剂的逻辑是：金融部门主要虹吸 AI/计算机/量化类人才，对人文、艺术类毕业生几无吸纳；因此若把被解释变量换成"非 AI 专业群供给 → 产业需求"构造的伪 CCD，金融虹吸系数应当消失。系数若不消失，说明 Siphon 捕捉的可能是某个城市层一般性混杂，而非"AI 人才再配置"机制。

二元安慰剂证伪了"机械产物"假说，但只用了"开/关"两个点。**剂量—反应安慰剂沿专业群的金融招聘强度梯度铺开多个点**：若虹吸机制为真，金融虹吸对耦合的负向效应应随该专业群被金融部门吸纳的强度**单调增强**——从高金融招聘的技术类（CS、量化、数理统计），递减到中间的其他理工、社科，直至几近于零的人文艺术类。一个单调梯度比"AI 显著、非 AI 为零"更难用巧合解释，是顶刊审稿人最认的"机制定向正确"证据。★

### （2）与 4.3.4 呼应：两条梯度的背离预测

更关键的是，剂量安慰剂可同时承担 4.3.4 的通道分离任务。对每个专业群 g，可计算两个群层属性：

- `FinHire_g`：该群向金融部门输送的强度（金融虹吸的"剂量"）
- `Expo_g`：该群的平均 LLM 远期暴露率（技术替代的"剂量"，取自魏立佳等 2026）

两条梯度给出**可分辨的背离预测**：

| | 金融虹吸机制（本文） | 技术替代机制（混杂） |
|---|---|---|
| 效应应跟随 | `FinHire_g` 梯度 | `Expo_g` 梯度 |
| 在"高暴露—低金融招聘"群上 | 效应弱 | 效应强 |
| 在"高金融招聘—低暴露"群上 | 效应强 | 效应弱 |

若实测效应沿 `FinHire_g` 单调而对 `Expo_g` 不敏感，则金融虹吸主导、技术替代被排除——这与 4.3.4 杠杆三（城市层赛马）从不同数据切面给出同一结论，构成三角互证。⚑

### （3）构造方法

**分组伪 CCD（逐群）：** 将全部本科专业按 `FinHire_g` 分为 K 个有序专业群（建议 K=4：①高金融招聘技术类 ②其他理工 ③社科财经类 ④人文艺术类）。对每个群 g：

- 用论文A 冻结口径下的逐专业招生计划，聚合出该群的 `E_index^g`（人均化、全样本固定基准 min-max、零值退化处理——一切跟随论文A 口径 ⛔ 不另设规则）；
- **I 端保持真实 I_index 不变**（企业存量+专利双模块），构造 `CCD^g`。该伪 CCD 度量"群 g 的教育供给与产业数字化需求的耦合"，对非 AI 群是人为耦合，但安慰剂检验的是 **Siphon 是否预测其变化**，而非其水平——故构造合法。

**逐群回归（生成梯度）：**

```
CCD^g_it = α + β₁·FinDev_{i,t-1} + β₂^g·Siphon_{i,t-1} + γ·X_it + μ_i + λ_t + ε_it
```

预期：`β₂^g` 随 `FinHire_g` 单调，g=① 最负、g=④ ≈ 0。画成"专业群 × 虹吸系数"梯度图作为正文图。

**堆叠交互（单一统计检验，最锐利）：** 将 K 个群层伪 CCD 堆叠为 城市×群×年 面板，做：

```
CCD_{igt} = α + β₁·FinDev_{i,t-1}
          + β_F·(Siphon_{i,t-1} × FinHire_g)
          + β_E·(Siphon_{i,t-1} × Expo_g)
          + δ_g + μ_i + λ_t + ε_{igt}
```

- 群固定效应 `δ_g` 吸收各群基线耦合水平差异；
- **判定**：`β_F < 0` 且显著、`β_E` 不显著/弱 → 效应定向跟随金融招聘而非技术暴露，金融虹吸机制成立、技术替代被排除。这是把 4.3.4 三杠杆压成一行系数对比的最锐版本。★

### （4）推断与设定纪律

- fractional logit 主报告 + 线性 FE 对照（同第4.1节）；解释变量滞后一期。
- wild cluster bootstrap（Webb 权重）+ Driscoll–Kraay 双报告（同第4.2节）。⚑ 堆叠面板按城市聚类，群维不构成独立聚类。
- ⛔ 所有伪 CCD 的标准化、零值、权重完全继承论文A 冻结口径，杜绝"为安慰剂另挑口径"。
- `Expo_g` 用三分位序数版另做一遍，规避魏文暴露率的循环偏差对基数的依赖。◎

### （5）数据可得性与诚实局限 ◆

| 要素 | 可得性 | 说明 |
|---|---|---|
| 各群 `E_index^g` | ◆高 | 论文A 逐专业招生计划可聚合到任意专业群 |
| `Expo_g` | ◆高 | 魏文 816 专业暴露率按群平均（需 B 口径专业→魏文专业映射表，纳入冻结文档附录）|
| `FinHire_g`（群→金融输送强度） | ◆中 | 依赖高校就业质量报告去向结构，口径不一 |

**局限（须正文写明）：**
1. `FinHire_g` 可得性为中——若群层去向数据不足，退一步用"专业是否属金融高吸纳类"的二元/三元粗分组，仍可生成弱梯度；最差情形回落到原二元安慰剂，结论不依赖剂量版。
2. 9 城市样本下逐群回归（列分群）功率有限，单调梯度可能不严格——故以堆叠交互的单一系数对比（β_F vs β_E）为主报告，逐群梯度图为辅助可视化。⚑ 扩样闽浙粤（WP0）显著改善功率。
3. `FinHire_g` 与 `Expo_g` 在专业层正相关（CS 两者皆高），但分组到群层、并辅以 4.3.4 杠杆一（时序分离）后，识别不依赖二者在群层完全正交。

### （6）与各模块的接口

- **与 4.3.4**：4.3.4 在主回归右端控制城市层暴露（`Expo_it`）；7.3 在被解释变量侧沿专业群梯度做安慰剂（`Expo_g`/`FinHire_g`）。同一混杂、两个数据切面、同一结论 → 三角互证。
- **与论文A**：所有 `E_index^g` 取自论文A 冻结口径，不重做测量（⚑）；映射表与分组规则写入冻结文档附录，事前锁定。
- **与第6章异质性**：剂量梯度若与第6.2节"虹吸主导城市的失调集中于 E供给—I岗位传导环节"的功能分解结论一致，则机制叙事在跨章层面自洽。
