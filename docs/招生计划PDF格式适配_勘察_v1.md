> 供 `scripts/parse_enrollment_pdf.py` 多版式适配留痕。各年版式不同，列坐标/布局/字体各异。

## 版式族（三族，勘察修订 2026-06-20）

| 年份 | 版式族 | 判别 | 解析状态 |
|---|---|---|---|
| 2022 / 2023 / 2024 / 2025 | **单栏·标准列带** | `detect_single_cols`→代号中位≥100 | ✅ 已适配（v2/v3） |
| **2021** | **单栏·左移列带**（无字体乱码） | `detect_single_cols`→代号中位<100 | ✅ 已适配（v4） |

| 2019 | 老高考版式（RAR：20190622*.rar，bsdtar 可解） | — | ⏭ 待勘察 |

> ⚠ 勘察更正：v1 误记"2020/2021 同为两栏乱码"。实测 **2021（物理组.pdf）是单栏、无 渊/冤 乱码，但列坐标整体左移**（专业代号@~97 落进 STD 的院校带、专业名称@~124 被切首字），用 STD 列带解析会全盘错位（福建校近乎零命中）。故拆为独立第三族。只有 **2020** 才是两栏+字体替换。

通用：水印"福建省教育考试院"为超大字号（67.6/52.2），过滤 `size<30` 即除；正文 ~8–9pt。

## 单栏列带（两套，按 `detect_single_cols` 选）

| 列 | COLS_STD（2022–2025） | COLS_2021（2021，左移） |
|---|---|---|
| 院校代号/校名 | 0–100 | 0–90 |
| 专业代号 | 100–130 | 90–112（数据@~97） |
| 专业名称 | 130–280 | 112–290（数据@~124） |
| 学制 | 280–300 | 290–312（数据@~303） |
| 计划人数 | 300–330 | 312–338（数据@~326） |
| 收费 | 330–362 | 338–366（数据@~348） |

判别：采样数据行 3 位专业代号 x 中位 <100 → 2021 左移带，否则标准带（`detect_single_cols`）。
专业=名称行 + 紧邻下方"代号 计划 收费"行配对（PAIR_TOL=14）。境外校区(林吉特/马来西亚)整段剔除。
2021 实测：莆田学院软件工程 plan=500 等大值经页内坐标核验为真实计划行（名称@402→数据@408），非错位；莆田软件大招生与 2022 一致（v2 docstring 已记"莆田软件306 为真"）。

## 2020/2021 两栏格式规范（勘察所得）

**字符替换映射**（该字体把符号渲染成生僻字，须先 translate）：
```
渊 → （      冤 → ）      袁 → ，      窑 → ·      尧 → 、
```
（员/园/造 等零星出现，低频，暂忽略或个案处理。）

**两栏切分**：page split ≈ x305；右栏相对左栏整体偏移 **+243**（右栏 x 减 243 后，两栏共用同一组左栏列带）。

**左栏列带（2020 实测）**：院校代号/校名 ~71–95 ｜ 专业代号 ~81–90（数据行）｜ 专业名称 ~93–235 ｜ 计划人数 ~240–250（数据行）｜ 学制 ~250–262（名称行）｜ 收费 ~265–290。
专业=名称行（名称@~99 + 学制@~253）+ 下方数据行（代号@~84 + 计划@~244 + 收费@~271）配对。

**已知坑（待解）**：
1. 左侧竖排批次标签（"高校农村专项计划"等单字竖排于 x26–37）干扰行解析 → 须剔除 x<60 的竖排单字。
2. 农村专项小段：数据行可能只有"代号 收费"、计划数错位/缺失 → 须核查该段计划数实际列位，或单独处理。
3. 大类招生：`计算机类（含…、人工智能）` 大类名可抽、按 broad_category_mapping→B；括号子专业为跨栏换行说明，不必解析。
4. 校名行/批次：批次以页头大字或竖排标签判；校名行含"公办/民办"。
