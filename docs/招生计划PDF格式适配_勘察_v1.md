## 单栏格式（2022–2025）列带（已用）

院校代号<100 ｜ 专业代号 100–130 ｜ 专业名称 130–280 ｜ 学制 280–300 ｜ 计划人数 300–330 ｜ 收费 330–362 ｜ 备注 >362。
专业=名称行 + 紧邻下方"代号 计划 收费"行配对。境外校区(林吉特/马来西亚)整段剔除。

## 2020/2021 两栏格式规范（勘察所得）

**字符替换映射**（该字体把符号渲染成生僻字，须先 translate）：
```
渊 → （      冤 → ）      袁 → ，      窑 → ·      尧 → 、
```
（员/园/造 等零星出现，低频，暂忽略或个案处理。）

**两栏切分**：page split ≈ x305；右栏相对左栏整体偏移 **+243**（右栏 x 减 243 后，两栏共用同一组左栏列带）。

**左栏列带（2020 实测）**：院校代号/校名 ~71–95 ｜ 专业代号 ~81–90（数据行）｜ 专业名称 ~93–235 ｜ 计划人数 ~240–250（数据行）｜ 学制 ~250–262（名称行）｜ 收费 ~265–290。
专业=名称行（名称@~99 + 学制@~253）+ 下方数据行（代号@~84 + 计划@~244 + 收费@~271）配对。

**已知坑（待解）**：
1. 左侧竖排批次标签（"高校农村专项计划"等单字竖排于 x26–37）干扰行解析 → 须剔除 x<60 的竖排单字。
2. 农村专项小段：数据行可能只有"代号 收费"、计划数错位/缺失 → 须核查该段计划数实际列位，或单独处理。
3. 大类招生：`计算机类（含…、人工智能）` 大类名可抽、按 broad_category_mapping→B；括号子专业为跨栏换行说明，不必解析。
4. 校名行/批次：批次以页头大字或竖排标签判；校名行含"公办/民办"。
