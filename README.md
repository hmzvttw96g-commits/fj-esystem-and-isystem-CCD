# Fujian E-I System CCD Experimental Analysis

This repository stores the normalized experimental analysis outputs, scripts, and documentation for the Fujian education-system and industry-system CCD pilot project.

## Scope

- Experimental CCD analysis for Fuzhou, Xiamen, and Quanzhou, 2019-2024.
- E-system recheck outputs, including Xiamen 2022 completion and Huaqiao University campus reassignment.
- I-system upgrade pilot materials for job and skill evidence.
- System-boundary audit outputs for E/I/F/G classification rules.

## Versioning Rules

- Raw data is not committed here.
- Legacy scattered outputs are not committed here.
- Formal outputs are stored under run_id directories.
- `latest_*` files are convenience entry points for the most recent reviewed result.
- CCD, calibration, E-system, I-system, and boundary-audit outputs remain experimental unless explicitly promoted in later documentation.

## Safety Notes

This repository intentionally excludes original PDFs, raw/source datasets, legacy outputs, virtual environments, caches, and temporary files.
