# O-ring 机制的"干净检验"缺口：要证 σ<1，需要什么数据

> 背景：辩论中 O-ring 的"方差放大"被实跑证伪（CV(D)=0.70 < 单环节 1.32），故 O-ring 当前只能作"**形态吻合**"（双峰 + 环节共动 corr≈0.90 成立），**不能**讲成"机制必然两极化"。要把它升级为机制，唯一干净的检验是估计四环节间的**替代弹性 σ**：O-ring/最弱环节⟺**互补**⟺ **σ<1**（极端 Leontief 时 σ→0）。本文说明该检验为何用现有数据做不了、以及需要补什么。

## 1. 为什么现有数据做不了（必须正视的循环陷阱）
- 你的 CCD = √(C·T)、C=2√(E·I)/(E+I) **已假定 E、I 相乘**；局部 CCD 同理。**用 CCD（或其分量）当被解释变量去"验证相乘" = 循环论证**，审稿人一眼识破。
- 四环节里"产业转化 = 专利"本身就是一种产出；用它当 Y、其余环节当投入，会与"产业转化"环节高度重叠，仍不干净。
- 结论：**必须引入一个独立于四环节的"外部产出 Y"**，由四环节共同生产，才能无偏地估 σ。

## 2. 需要补的"外部产出 Y"候选（城市×年面板，独立于四环节）
按优先级（越靠前越干净、越可得）：
1. **数字经济/信息产业增加值**（或规上工业数字化部分增加值）——最贴近"人才→产出"，且不等于专利或从业人数。来源：省/市统计年鉴、数字经济发展报告。
2. **新建 AI/数字技术企业数 或 风投/融资额**（企业形成）——反映人才落地为经济活动；来源：企查查/天眼查类工商登记、CVSource/IT桔子类创投库。
3. **技术合同成交额 / 技术市场交易**——反映创新转化为交易；来源：科技统计年鉴。
4. **全要素生产率（TFP）或劳动生产率**——经典产出；可由统计年鉴产出+投入估算。
5. **AI 相关岗位薪资 或 招聘填补率**——劳动力市场产出侧；来源：招聘平台报告（须核口径）。

> 选取准则：Y 必须 (a) 城市×年可得、(b) 口径独立于"招生/一流/门类I从业/专利"四环节、(c) 语义上是"人才—产业匹配的下游成果"。

## 3. 估计设计（拿到 Y 后）
- **CES 生产函数**：Y = A·[Σ_k α_k · L_k^ρ]^{1/ρ}，σ = 1/(1−ρ)。检验 **σ<1（ρ<0）= 互补/O-ring**；σ→0 = Leontief（极端最弱环节）。
- **降维（关键）**：4 投入 CES 在 9 市×6 年=54 观测上**过度参数化**。两条出路：
  - (a) **嵌套 2 投入**：先估 E、I 两系统间的 σ（最关心的"教育—产业"互补），再在系统内估 σ；
  - (b) **扩样本到全国地级市**（~290×多年）——这也顺带解决你一直的 N 问题。
- **稳健替代**：translog 生产函数 + 检验投入间 **Allen–Uzawa 替代弹性 / 交叉偏导符号**（互补 ⟺ 交叉偏导>0）。
- **识别**：投入内生（Y 高的地方更吸引教育与产业投入）→ 需投入的工具（历史高校布局、Bartik 需求移位份额等，见辩论文档 Ⅳ.3 / 因果设计）。**故"干净 σ 检验"本身也带内生性，属扩展数据 + 因果工具的工作。**

## 4. 在补到数据之前，O-ring 该怎么写（诚实降级）
- 写成 **"形态学证据 + 待检机制"**：报告 (a) D 的**双峰**（核心簇 vs 腹地簇、中段稀疏）、(b) 四环节**强共动**（corr≈0.90，支持同配/互补），作为 O-ring **一致的典型事实**；
- **明确不声称** σ<1 / "方差放大"（后者已被本数据证伪）；
- 把 σ<1 的 CES/translog 检验列为**可检验假设 + 数据需求**（本规格即其蓝图），留待补入"外部产出 Y"（最好同时扩到全国样本）后执行。

## 5. 一句话给导师
> O-ring 现在是"长得像"（双峰+共动），还不是"证明了"（σ<1 未测、方差放大已被证伪）；要证明，需要一个独立于四环节的城市级产出 Y，并最好扩到全国样本、配投入工具——这是明确的下一步，不是现在能下的结论。
