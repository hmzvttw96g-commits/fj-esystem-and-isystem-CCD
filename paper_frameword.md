# 福建省AI教育—产业—金融耦合研究总路线图

本仓库目前设立两条并列研究线：

1. **课题论文系列：教育系统—产业系统双系统研究**
2. **毕业论文系列：教育系统—产业系统—金融系统三系统研究**

两条研究线共享同一套基础逻辑：

> AI教育供给 → AI人才与知识产出 → AI产业吸收 → 技术转化与产业承接 → 科技金融支持 → 资本回流与再投入

当前导师课题论文先研究前半段：

> AI教育供给 → AI产业承接

未来金融专业硕士毕业论文在此基础上加入科技金融系统，研究完整功能循环：

> AI教育供给 → AI产业承接 → 科技金融支持 → 资本回流与再投入

## 二、总体研究逻辑

本研究不再把教育、产业和金融看作静态并列系统，而是将其放入AI功能链中理解。

| 功能链环节 | 对应系统 | 核心含义 |
|---|---|---|
| 人才与知识供给 | 教育系统 E | 高校、专业、招生、平台提供AI人才与知识源头 |
| 人才吸收与技术转化 | 产业系统 I | 企业、岗位、高企、专利承接教育端供给 |
| 资本支持与风险分担 | 金融系统 F | 科技贷款、知识产权质押、创投和政府基金支持转化 |
| 资本回流与再投入 | 金融—产业—教育反馈 | 金融资源反过来支持企业研发、平台建设和产教融合 |

这一设计的核心意义是：

> 研究福建省AI发展的关键，不是简单比较高校数量、企业数量或贷款规模，而是识别AI人才、技术、产业和资本之间的功能循环是否顺畅。

## 三、课题论文指标设计：教育—产业双系统

### 1. 教育系统 E：AI人才与知识供给能力

| 维度 | 指标 | 当前处理意见 |
|---|---|---|
| 人才培养入口 | AI核心专业数量 | 主模型优先保留 |
| 培养主体 | 开设AI核心专业高校数量 | 主模型保留，但避免单独作为核心判断 |
| 人才供给规模 | AI相关专业招生计划人数 | 数据可得则优先保留 |
| 知识与平台供给 | AI重点实验室、产业学院、产教融合平台数量 | 拓展变量或机制变量 |
| 知识产出 | 高校AI论文、高校AI专利 | 可作为稳健性或补充指标 |

### 2. 产业系统 I：AI产业吸收与承接能力

| 维度 | 指标 | 当前处理意见 |
|---|---|---|
| 产业主体 | AI相关企业数量 | 主模型保留 |
| 产业质量 | AI相关高新技术企业数量 | 主模型保留 |
| 人才吸收 | AI岗位需求数量 | 关键连接指标，优先收集 |
| 技术转化 | 企业AI专利数量 | 主模型保留 |
| 产业活力 | 新增AI企业数量 | 拓展或稳健性指标 |

课题论文的核心被解释对象建议为：

- 教育系统综合指数 E；
- 产业系统综合指数 I；
- 教育—产业耦合协调度 \(D_{EI}\)；
- 教育—产业错配指数 \(E-I\)。

## 四、毕业论文指标设计：教育—产业—金融三系统

毕业论文沿用课题论文中的E系统和I系统，并新增科技金融支持系统F。

### 金融系统 F：科技金融支持与资本回流能力

| 维度 | 指标 | 当前处理意见 |
|---|---|---|
| 研发资金支持 | 科技型企业贷款余额 | 主模型优先保留 |
| 知识资产金融化 | 知识产权质押融资金额或项目数 | 主模型优先保留 |
| 风险资本支持 | 创投事件数或创投金额 | 数据可得则保留 |
| 政府引导支持 | 政府引导基金项目数或规模 | 数据可得则保留 |
| 金融环境 | 数字普惠金融指数、贷款余额/GDP | 控制变量或稳健性变量，不替代科技金融主变量 |

毕业论文的核心被解释对象和解释变量建议为：

- 教育—产业耦合协调度 \(D_{EI}\)；
- 教育—产业—金融三系统协调度 \(D_{EIF}\)；
- 科技金融支持指数 F；
- 科技金融对 \(D_{EI}\) 的影响。

---

## 五、模型路线

### 1. 课题论文主模型

1. 构建教育系统综合指数 E；
2. 构建产业系统综合指数 I；
3. 测算教育—产业耦合协调度 \(D_{EI}\)；
4. 识别教育超前、产业超前、双强协调、双弱滞后四类城市；
5. 若找到合适政策冲击，进一步进行DID、强度型DID或DDD检验。

### 2. 毕业论文主模型

1. 沿用课题论文中的E、I与 \(D_{EI}\)；
2. 新增科技金融支持指数 F；
3. 测算三系统耦合协调度 \(D_{EIF}\)；
4. 使用面板回归检验科技金融是否提升 \(D_{EI}\)；
5. 做异质性、稳健性和障碍度分析。

## 六、后续指标迭代原则

之后所有指标设计都按照以下原则动态调整：

1. **功能链优先**：指标必须能对应“供给—吸收—转化—支持—反馈”的某一环节；
2. **可获得优先**：主模型只放连续年份、城市层面可获得的数据；
3. **可复现优先**：指标必须说明来源、口径、关键词、爬取规则或统计规则；
4. **主变量克制**：难以核验的数据不进入主模型，只做补充说明；
5. **金融系统暂不进入课题论文主模型**：课题论文只写教育—产业，金融系统保留给毕业论文；
6. **同一指标不得重复归类**：融资、贷款、基金、质押等金融变量不得放入产业系统；高校、招生、专业等教育变量不得放入产业系统。
