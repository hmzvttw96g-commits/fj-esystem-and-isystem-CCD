# 公式一：商润泽(2024) 二维CCD公式

## 适用场景
- 研究对象：**两个子系统**（如教育 vs 产业）
- 论文A（AI学科-产业专项耦合）采用此公式

## 适用场景
- 研究对象：**两个子系统**（如教育 vs 产业）
- 论文A（AI学科-产业专项耦合）采用此公式

## 计算步骤

### Step 1：子系统评价指数

```
J(A) = Σ(教育指标 × 权重)    ← 科类结构评价函数
 g(B) = Σ(产业指标 × 权重)   ← 产业结构评价函数
```

### Step 2：耦合度

```
           J(A) × g(B)
C = ───────────────────────
     [J(A) + g(B)]²
```

### Step 3：综合评价指数

```
T = α × J(A) + β × g(B)

其中 α = β = 0.5（两子系统同等重要）
```

### Step 4：耦合协调度（最终输出）

```
        ┌─────────────┐
        │    C  ×  T  │
D = √   │    ↓       │
        │   开根号    │
        └─────────────┘
```

## 耦合度分级标准

| C值 | 耦合层次 |
|-----|---------|
| 0 < C ≤ 0.3 | 低水平耦合 |
| 0.3 < C ≤ 0.5 | 耦合颉颃 |
| 0.5 < C ≤ 0.8 | 耦合磨合 |
| 0.8 < C ≤ 1 | 高水平耦合 |

## 耦合协调度分级标准

| D值 | 协调层次 |
|-----|---------|
| 0 < D ≤ 0.2 | 失调衰退 |
| 0.2 < D ≤ 0.4 | 低度协调 |
| 0.4 < D ≤ 0.6 | 中度协调 |
| 0.6 < D ≤ 0.8 | 高度协调 |
| 0.8 < D ≤ 1 | 极度协调 |

# 公式二：东千宇(2023) 三维CCD乘积形式

## 适用场景
- 研究对象：**三个子系统**（如能源 vs 经济 vs 环境）
- 三系统存在**相互促进**关系

## 计算步骤

### Step 1：子系统评价指数

```
f₁ = Σ(能源指标 × 权重)     ← 能源子系统
f₂ = Σ(经济指标 × 权重)     ← 经济子系统
f₃ = Σ(环境指标 × 权重)     ← 环境子系统
```

### Step 2：三维耦合度

```
           f₁ × f₂ × f₃
C = ───────────────────────
     (f₁ + f₂ + f₃)³
```

### Step 3：综合评价指数

```
T = (f₁ + f₂ + f₃) ÷ 3
```

### Step 4：耦合协调度

```
        ┌─────────────┐
        │    C  ×  T  │
D = √   │    ↓       │
        │   开根号    │
        └─────────────┘
```

## ⚠️ 问题：数值验证

```
设：f₁=0.7  f₂=0.5  f₃=0.6

C = (0.7×0.5×0.6) / (0.7+0.5+0.6)³
  = 0.21 / 5.832
  = 0.036

T = (0.7+0.5+0.6) ÷ 3 = 0.6

D = √(0.036 × 0.6) = 0.147

❌ 问题：D=0.147 判为"严重失调"
   但三个系统都中等偏上，明显不合理！
```

# 公式三：两两耦合加权平均（推荐！）

## 适用场景
- 研究对象：**三个子系统**（金融 vs 教育 vs 产业）
- 论文D（金融-教育-产业三维耦合）采用此公式
- **继承商润泽二维公式**，稳健易解释

## 计算步骤

### Step 1：子系统评价指数

```
F = Σ(金融指标 × 权重)     ← 金融子系统
E = Σ(教育指标 × 权重)     ← 教育子系统
I = Σ(产业指标 × 权重)     ← 产业子系统
```

### Step 2：两两耦合度（计算3个）

```
         2 × F × E
C_FE = ───────────
        (F + E)²
        ↑
     金融-教育耦合

         2 × E × I
C_EI = ───────────
        (E + I)²
        ↑
     教育-产业耦合

         2 × F × I
C_FI = ───────────
        (F + I)²
        ↑
     金融-产业耦合
```

### Step 3：三维综合耦合度

```
C_3D = (C_FE + C_EI + C_FI) ÷ 3
        ↑
     三个两两耦合度的算术平均
```

### Step 4：综合评价指数

```
T = (F + E + I) ÷ 3
```

### Step 5：三维耦合协调度（最终输出）

```
        ┌─────────────┐
        │  C_3D  ×  T  │
D_3D = √│   ↓        │
        │   开根号    │
        └─────────────┘
```

## 计算流程图

```
          ┌─────────────────────────────────────┐
          │         三个子系统得分              │
          │   F（金融） E（教育） I（产业）      │
          └──────────┬──────────────────────────┘
                     │
          ┌──────────▼──────────────────────────┐
          │      两两耦合度（3个）              │
          │  C_FE    C_EI    C_FI              │
          └──────────┬──────────────────────────┘
                     │
          ┌──────────▼──────────────────────────┐
          │   C_3D = (C_FE+C_EI+C_FI)÷3       │
          │        三维综合耦合度                │
          └──────────┬──────────────────────────┘
                     │
          ┌──────────▼──────────────────────────┐
          │   T = (F+E+I)÷3                    │
          │      综合评价指数                    │
          └──────────┬──────────────────────────┘
                     │
          ┌──────────▼──────────────────────────┐
          │   D_3D = √(C_3D × T)              │
          │      最终输出：耦合协调度             │
          └─────────────────────────────────────┘
```

## 数值验证

```
设：F=0.7  E=0.5  I=0.6

C_FE = 2×0.7×0.5 / (0.7+0.5)² = 0.7/1.44 = 0.486
C_EI = 2×0.5×0.6 / (0.5+0.6)² = 0.6/1.21 = 0.496
C_FI = 2×0.7×0.6 / (0.7+0.6)² = 0.84/1.69 = 0.497

C_3D = (0.486 + 0.496 + 0.497) ÷ 3 = 0.493

T = (0.7+0.5+0.6) ÷ 3 = 0.6

D_3D = √(0.493 × 0.6) = √0.296 = 0.544

✅ 结果：D_3D=0.544 → 勉强协调（符合实际）
```

# 公式四：几何平均×离散修正

## 适用场景
- 研究对象：**三个子系统**，但更强调**系统间均衡性**
- 理论更精确，但计算相对复杂

## 计算步骤

### Step 1-2：同公式三

```
F, E, I → C_FE, C_EI, C_FI
```

### Step 3：几何平均

```
              ────────────────
C_几何 = ³√ C_FE × C_EI × C_FI
         ↑
      三个耦合度的几何平均
```

### Step 4：离散程度

```
δ = |F - E| × |E - I| × |F - I|
  ↑
三系统两两差距的乘积
δ越小 = 三系统越均衡
```

### Step 5：离散修正后的三维耦合度

```
C_3D = C_几何 × (1 - δ)
         ↑
      离散程度越大，耦合度折扣越多
```

### Step 6-7：同公式三

```
T = (F + E + I) ÷ 3
D_3D = √(C_3D × T)
```

## 数值验证

```
设：F=0.7  E=0.5  I=0.6

C_FE = 0.486
C_EI = 0.496
C_FI = 0.497

C_几何 = ³√(0.486 × 0.496 × 0.497) = ³√0.120 = 0.493

δ = |0.7-0.5| × |0.5-0.6| × |0.7-0.6|
  = 0.2 × 0.1 × 0.1 = 0.002

C_3D = 0.493 × (1 - 0.002) = 0.492

T = 0.6

D_3D = √(0.492 × 0.6) = √0.295 = 0.543

✅ 结果：D_3D=0.543 → 勉强协调
   与公式三接近，但额外考虑了系统均衡性
```

# Python实现代码

```python
import numpy as np

def ccd_2d(J_A, g_B, alpha=0.5, beta=0.5):
    """商润泽(2024)二维CCD公式"""
    C = (J_A * g_B) / ((J_A + g_B) ** 2)
    T = alpha * J_A + beta * g_B
    D = np.sqrt(C * T)
    return C, D

def ccd_3d_weighted(F, E, I):
    """公式三：两两耦合加权平均"""
    # 两两耦合度
    C_FE = 2 * F * E / ((F + E) ** 2)
    C_EI = 2 * E * I / ((E + I) ** 2)
    C_FI = 2 * F * I / ((F + I) ** 2)
    
    # 三维综合耦合度
    C_3D = (C_FE + C_EI + C_FI) / 3
    
    # 综合评价指数
    T = (F + E + I) / 3
    
    # 耦合协调度
    D_3D = np.sqrt(C_3D * T)
    
    return C_3D, D_3D

def ccd_3d_geometry(F, E, I):
    """公式四：几何平均+离散修正"""
    # 两两耦合度
    C_FE = 2 * F * E / ((F + E) ** 2)
    C_EI = 2 * E * I / ((E + I) ** 2)
    C_FI = 2 * F * I / ((F + I) ** 2)
    
    # 几何平均
    C_geo = (C_FE * C_EI * C_FI) ** (1/3)
    
    # 离散程度
    delta = abs(F - E) * abs(E - I) * abs(F - I)
    
    # 离散修正
    C_3D = C_geo * (1 - delta)
    
    # 综合评价指数
    T = (F + E + I) / 3
    
    # 耦合协调度
    D_3D = np.sqrt(C_3D * T)
    
    return C_3D, D_3D
```
