### 2.1 核心公式

```
                    ┌─────────────────┐

                    └────────┬────────┘
                             │
              ┌─────────────┼─────────────┐
```

```
步骤1：计算 E 和 I
   E = +0.033
   I = +0.023

   Δ = 0.033 - 0.023 = +0.010

步骤3：判断类型
   +0.010 ∈ (-0.1, 0] → 学科略优型

结论：福建省AI学科略优于AI产业
```

## 三、公式汇总

### 3.1 标准化公式

```
      xᵢ - x̄
xᵢᵢ = ────────
     xₘₐₓ - x̄

其中：
  xᵢ  = 福建省指标值
  x̄   = 全国平均水平
  xₘₐₓ = 全国最大值
```

### 3.2 子系统评价指数公式

```
   ┌─────────────────────────────────────────┐
   │                                         │
   │   F = Σ(wFj × xFj_std)                  │
   │   E = Σ(wEj × xEj_std)                  │
   │   I = Σ(wIj × xIj_std)                  │
   │                                         │
   └─────────────────────────────────────────┘
```

### 3.3 两两耦合度公式

```
         2 × F × E                2 × E × I                2 × F × I
C_FE = ───────────        C_EI = ───────────        C_FI = ───────────
       (F + E)²                  (E + I)²                  (F + I)²
```

### 3.4 三维综合耦合度公式

```
                 ┌───────────────────────────┐
                 │  C_FE + C_EI + C_FI      │
         C_3D = │  ────────────────────     │
                 │           3              │
                 └───────────────────────────┘
```

### 3.5 耦合协调度公式

```
   ┌────────────────────────────────────────┐
   │                                        │
   │   T = (F + E + I) / 3                  │
   │                                        │
   │   D_3D = √(C_3D × T)                   │
   │                                        │
   └────────────────────────────────────────┘
```

### 3.6 相对地位判断公式

```
      Δ = E - I

      > +0.1  →  学科超前型
      0~+0.1  →  学科略优型  ← 福建省
     -0.1~0   →  产业略优型
      < -0.1  →  产业超前型
```

## 七、公式速查表

| 公式名称 | 公式 | 用途 |
|---------|------|------|
| 横向标准化 | x_std = (x - x̄)/(x_max - x̄) | 将指标转为相对分数 |
| 子系统得分 | F/E/I = Σ(w × x_std) | 计算各系统综合得分 |
| 两两耦合度 | C_ab = 2ab/(a+b)² | 衡量两系统交互强度 |
| 三维耦合度 | C_3D = (C_FE+C_EI+C_FI)/3 | 三系统综合耦合 |
| 综合评价 | T = (F+E+I)/3 | 三系统整体水平 |
| 耦合协调度 | D_3D = √(C_3D × T) | 最终输出指标 |
| 相对地位 | Δ = E - I | 判读学科vs产业 |
