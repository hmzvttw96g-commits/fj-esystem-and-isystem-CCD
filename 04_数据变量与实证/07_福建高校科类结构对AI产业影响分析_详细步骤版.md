## 第三步：标准化处理

### 3.1 标准化公式

由于各指标量纲不同，需要将原始数据标准化到[0,1]区间：

$$a_i^{std} = \frac{a_i - a_{min}}{a_{max} - a_{min}}$$

$$b_j^{std} = \frac{b_j - b_{min}}{b_{max} - b_{min}}$$

### 3.2 教育指标标准化

| 教育指标 | 原始值 | 最大值参考 | 最小值参考 | 标准化值 |
|---------|-------|-----------|-----------|---------|
| a₁ | 25% | 40% | 10% | (25-10)/(40-10) = 0.500 |
| a₂ | 32.5% | 40% | 20% | (32.5-20)/(40-20) = 0.625 |
| a₃ | 7.5% | 15% | 5% | (7.5-5)/(15-5) = 0.250 |
| a₄ | 23.8% | 30% | 15% | (23.8-15)/(30-15) = 0.587 |
| a₅ | 11.2% | 20% | 5% | (11.2-5)/(20-5) = 0.413 |

### 3.3 产业指标标准化

| 产业指标 | 原始值 | 最大值参考 | 最小值参考 | 标准化值 |
|---------|-------|-----------|-----------|---------|
| b₁ | 35% | 40% | 20% | (35-20)/(40-20) = 0.750 |
| b₂ | 25% | 35% | 15% | (25-15)/(35-15) = 0.500 |
| b₃ | 15% | 20% | 8% | (15-8)/(20-8) = 0.583 |
| b₄ | 12% | 15% | 5% | (12-5)/(15-5) = 0.700 |
| b₅ | 10% | 15% | 5% | (10-5)/(15-5) = 0.500 |

## 第四步：计算教育系统评价指数 J(A)

### 4.1 计算公式

$$J(A) = \sum_{i=1}^{5} w_i \times a_i^{std}$$

### 4.2 代入计算

```
J(A) = w₁ × a₁ + w₂ × a₂ + w₃ × a₃ + w₄ × a₄ + w₅ × a₅

J(A) = 0.40 × 0.500 
     + 0.25 × 0.625 
     + 0.15 × 0.250 
     + 0.10 × 0.587 
     + 0.10 × 0.413
```

### 4.3 分步计算

```
第一项：0.40 × 0.500 = 0.200

第二项：0.25 × 0.625 = 0.156

第三项：0.15 × 0.250 = 0.038

第四项：0.10 × 0.587 = 0.059

第五项：0.10 × 0.413 = 0.041
```

### 4.4 求和结果

$$J(A) = 0.200 + 0.156 + 0.038 + 0.059 + 0.041 = 0.494$$

**教育系统评价指数 J(A) = 0.494**

---

## 第五步：计算产业系统评价指数 g(B)

### 5.1 计算公式

$$g(B) = \sum_{j=1}^{5} w_j \times b_j^{std}$$

### 5.2 代入计算

```
g(B) = w₁ × b₁ + w₂ × b₂ + w₃ × b₃ + w₄ × b₄ + w₅ × b₅

g(B) = 0.35 × 0.750 
     + 0.30 × 0.500 
     + 0.15 × 0.583 
     + 0.12 × 0.700 
     + 0.08 × 0.500
```

### 5.3 分步计算

```
第一项：0.35 × 0.750 = 0.263

第二项：0.30 × 0.500 = 0.150

第三项：0.15 × 0.583 = 0.087

第四项：0.12 × 0.700 = 0.084

第五项：0.08 × 0.500 = 0.040
```

### 5.4 求和结果

$$g(B) = 0.263 + 0.150 + 0.087 + 0.084 + 0.040 = 0.624$$

**产业系统评价指数 g(B) = 0.624**

---

## 第六步：计算耦合度 C

### 6.1 耦合度公式（商润泽2024）

$$C = \frac{J(A) \times g(B)}{[J(A) + g(B)]^2}$$

### 6.2 代入已知数值

```
已知：J(A) = 0.494
      g(B) = 0.624

则：
分子 = J(A) × g(B) = 0.494 × 0.624

分母 = [J(A) + g(B)]² = [0.494 + 0.624]² = [1.118]²
```

### 6.3 分步计算

```
第一步：计算分子
分子 = 0.494 × 0.624 = 0.308

第二步：计算分母的和
J(A) + g(B) = 0.494 + 0.624 = 1.118

第三步：计算分母的平方
分母 = 1.118² = 1.118 × 1.118 = 1.250

第四步：计算耦合度
C = 0.308 / 1.250 = 0.246
```

### 6.4 耦合度结果

$$C = \frac{0.494 \times 0.624}{(0.494 + 0.624)^2} = \frac{0.308}{1.250} = 0.246$$

**耦合度 C = 0.246**

---

## 第七步：计算综合评价指数 T

### 7.1 综合评价指数公式

$$T = \alpha \times J(A) + \beta \times g(B)$$

其中 α = β = 0.5（教育与产业同等重要）

### 7.2 代入计算

```
T = 0.5 × J(A) + 0.5 × g(B)
T = 0.5 × 0.494 + 0.5 × 0.624
```

### 7.3 分步计算

```
第一项：0.5 × 0.494 = 0.247

第二项：0.5 × 0.624 = 0.312

求和：T = 0.247 + 0.312 = 0.559
```

### 7.4 综合评价指数结果

$$T = 0.5 \times 0.494 + 0.5 \times 0.624 = 0.559$$

**综合评价指数 T = 0.559**

---

## 第八步：计算耦合协调度 D

### 8.1 耦合协调度公式

$$D = \sqrt{C \times T}$$

### 8.2 代入计算

```
D = √(C × T)
D = √(0.246 × 0.559)
```

### 8.3 分步计算

```
第一步：计算乘积
C × T = 0.246 × 0.559 = 0.138

第二步：开平方根
D = √0.138 = 0.371
```

### 8.4 耦合协调度结果

$$D = \sqrt{0.246 \times 0.559} = \sqrt{0.138} = 0.371$$

**耦合协调度 D = 0.371**

---

## 第九步：等级判定

### 9.1 耦合协调度分级标准

| D值区间 | 等级 | 含义 |
|--------|------|------|
| 0.8 < D ≤ 1.0 | 优质协调 | 教育与产业高度匹配 |
| 0.6 < D ≤ 0.8 | 良好协调 | 教育与产业较好匹配 |
| 0.4 < D ≤ 0.6 | 初级协调 | 教育与产业基本匹配 |
| 0.2 < D ≤ 0.4 | **濒临失调** | 教育与产业存在偏差 |
| 0.0 < D ≤ 0.2 | 严重失调 | 教育与产业严重脱节 |

### 9.2 判定结果

```
D = 0.371

判定：D = 0.371 落在 [0.2, 0.4] 区间

结论：福建省高校科类结构与AI产业人才需求
     处于"濒临失调"等级
```
