```python
import numpy as np

fig, ax = plt.subplots(figsize=(10, 6))
```

## 二、空间演变分析

### 2.1 全局空间自相关

**Moran's I 指数**：
$$I = \frac{n}{W} \times \frac{\sum_i \sum_j w_{ij}(x_i - \bar{x})(x_j - \bar{x})}{\sum_i (x_i - \bar{x})^2}$$

```python
from libpysal.weights import Queen
from esda.moran import Moran, Moran_Local
import geopandas as gpd

# 读取地理数据
gdf = gpd.read_file('china_province.shp')

# 构建空间权重矩阵
w = Queen.from_dataframe(gdf)
w.transform = 'r'

# 计算全局Moran's I
y = df['coupling_degree'].values
moran = Moran(y, w)
print(f"Moran's I: {moran.I:.4f}")
print(f"P-value: {moran.p_sim:.4f}")
```

### 2.2 局部空间自相关（LISA）

```python
from esda.moran import Moran_Local

# 计算局部Moran's I
lisa = Moran_Local(y, w)
```

## 三、收敛性分析

### 3.1 σ收敛

检验区域间差异是否随时间缩小：
$$\sigma_t = \sqrt{\frac{1}{n} \sum_{i=1}^{n}(D_{it} - \bar{D}_t)^2}$$

### 3.2 β收敛

检验落后地区是否追赶发达地区：
$$ln(D_{i,t}/D_{i,t-1}) = \alpha + \beta ln(D_{i,t-1}) + \varepsilon$$

```python
import statsmodels.api as sm

# β收敛检验
df['ln_growth'] = np.log(df['D'].shift(-1) / df['D'])
df['ln_D_lag'] = np.log(df['D'])

X = sm.add_constant(df['ln_D_lag'].dropna())
y = df['ln_growth'].dropna()

model = sm.OLS(y.iloc[:len(X)], X.dropna())
results = model.fit()
print(results.summary())
```
