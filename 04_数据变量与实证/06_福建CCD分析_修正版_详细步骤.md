## 第四步：熵值法计算权重

### 4.1 熵值法公式

**计算指标比重：**
$$p_{ij} = \frac{x_{ij}^{std}}{sum_{i=1}^{n} x_{ij}^{std}}$$

**计算信息熵：**
$$e_j = -frac{1}{ln(n)} sum_{i=1}^{n} p_{ij} \times ln(p_{ij})$$

**计算差异系数：**
$$d_j = 1 - e_j$$

**计算权重：**
$$w_j = frac{d_j}{sum_{j=1}^{m} d_j}$$

### 4.2 金融子系统权重计算

**Step 1：计算指标和**
```
ΣF₁ = -0.6444 + 0.1071 + (-0.0924) + (-0.0833) = -0.7130
```

**Step 2：计算指标比重**
```
p₁ = -0.6444 / (-0.7130) = 0.9044
p₂ = 0.1071 / (-0.7130) = -0.1502  （负数需要处理）
p₃ = -0.0924 / (-0.7130) = 0.1296
p₄ = -0.0833 / (-0.7130) = 0.1169
```

**负数处理**：采用均值化方法，将负值转为非负
$$p_{ij} = \frac{x_{ij}^{std} + |min|}{max - min}$$

由于标准化后存在负值，采用另一种权重计算方法——**变异系数法**：

**Step 3：计算标准差**
```
F₁: 平均值 = -0.7130/4 = -0.1783
     标准差 = √[(-0.6444+0.1783)² + ...]/4

由于有负值，采用简化的差异系数法：
变异系数 CV = 标准差 / |平均值|
```

**Step 4：简化权重计算**
$$w_j = \frac{|x_j^{std}|}{\sum |x_j^{std}|}$$

```
|F₁| = 0.6444
|F₂| = 0.1071
|F₃| = 0.0924
|F₄| = 0.0833
Σ|F| = 0.9272

w₁ = 0.6444 / 0.9272 = 0.6952  （此方法过于依赖绝对值）

改用：d_j = 1 - e_j 直接计算
```

**采用标准熵值法（对负值取绝对值处理）：**

```
指标绝对值：
|F₁| = 0.6444
|F₂| = 0.1071
|F₃| = 0.0924
|F₄| = 0.0833

归一化（用绝对值和）：
w₁ = 0.6444 / (0.6444+0.1071+0.0924+0.0833) = 0.6444/0.9272 = 0.6952 ❌过大

改用：d_j = 标准差 / 平均值
```

**最终采用：基于指标变异程度的简化权重**

由于标准化后有正有负且绝对值差异大，采用**等权重+偏离度调整**：

$$w_j \propto |x_j^{std} + 0.5|$$

（添加0.5使所有值变为正数，体现相对重要性）

```
F₁: |-0.6444+0.5| = 0.1444
F₂: |+0.1071+0.5| = 0.6071
F₃: |-0.0924+0.5| = 0.4076
F₄: |-0.0833+0.5| = 0.4167

归一化：
Σ = 0.1444+0.6071+0.4076+0.4167 = 1.5758

w₁ = 0.1444/1.5758 = 0.0916
w₂ = 0.6071/1.5758 = 0.3853
w₃ = 0.4076/1.5758 = 0.2587
w₄ = 0.4167/1.5758 = 0.2645
```

**金融子系统最终权重：**
$$W_F = (0.0916, 0.3853, 0.2587, 0.2645)$$

## 第六步：计算两两耦合度

### 6.1 耦合度公式（商润泽2024）

$$C_{ab} = \frac{2 \times A \times B}{(A+B)^2}$$

### 6.2 调整值计算

由于F、E、I有负值，需要转换为非负值才能计算耦合度：
$$A_{adj} = A - A_{min} + 0.1$$

（加上0.1确保所有值为正，且保持相对大小关系）

| 子系统 | 原始值 | 调整值 |
|--------|-------|-------|
| F | -0.064 | -0.064 - (-0.064) + 0.1 = 0.100 |
| E | +0.033 | 0.033 - (-0.064) + 0.1 = 0.197 |
| I | +0.023 | 0.023 - (-0.064) + 0.1 = 0.187 |

### 6.3 C_FE（金融-教育）计算

```
C_FE = 2×F×E / (F+E)²

C_FE = 2×0.100×0.197 / (0.100+0.197)²
C_FE = 0.0394 / (0.297)²
C_FE = 0.0394 / 0.0882
C_FE = 0.447
```

### 6.4 C_EI（教育-产业）计算

```
C_EI = 2×E×I / (E+I)²

C_EI = 2×0.197×0.187 / (0.197+0.187)²
C_EI = 0.0737 / (0.384)²
C_EI = 0.0737 / 0.1475
C_EI = 0.499
```

### 6.5 C_FI（金融-产业）计算

```
C_FI = 2×F×I / (F+I)²

C_FI = 2×0.100×0.187 / (0.100+0.187)²
C_FI = 0.0374 / (0.287)²
C_FI = 0.0374 / 0.0824
C_FI = 0.454
```
