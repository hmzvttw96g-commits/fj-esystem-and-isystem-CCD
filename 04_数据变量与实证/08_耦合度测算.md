# 耦合协调度测算

## 一、测算模型

### 1.1 耦合度模型

$$C = \sqrt{\frac{u_1 \times u_2}{(\frac{u_1 + u_2}{2})^2}} = \frac{2\sqrt{u_1 \times u_2}}{u_1 + u_2}$$

### 1.2 综合评价指数

$$T = \alpha u_1 + \beta u_2$$

其中：$\alpha = \beta = 0.5$（等权重）

### 1.3 耦合协调度

$$D = \sqrt{C \times T}$$

## 二、Python实现

```python
import pandas as pd
import numpy as np

def calculate_coupling(u1, u2):
    """计算耦合度"""
    return 2 * np.sqrt(u1 * u2) / (u1 + u2)

def calculate_coordination(u1, u2, alpha=0.5, beta=0.5):
    """计算综合评价指数"""
    return alpha * u1 + beta * u2

def calculate_coupling_degree(u1, u2, alpha=0.5, beta=0.5):
    """计算耦合协调度"""
    C = calculate_coupling(u1, u2)
    T = calculate_coordination(u1, u2, alpha, beta)
    D = np.sqrt(C * T)
    return C, T, D

def classify_coupling(D):
    """划分耦合协调等级"""
    if D < 0.2:
        return "严重失调"
    elif D < 0.3:
        return "中度失调"
    elif D < 0.4:
        return "轻度失调"
    elif D < 0.5:
        return "濒临失调"
    elif D < 0.6:
        return "勉强协调"
    elif D < 0.7:
        return "初级协调"
    elif D < 0.8:
        return "中级协调"
    elif D < 0.9:
        return "良好协调"
    else:
        return "优质协调"
```

## 三、熵权法确定权重

```python
def entropy_weight(data):
    """熵权法计算权重"""
    # 标准化
    data_norm = (data - data.min()) / (data.max() - data.min())
    
    # 计算比重
    p = data_norm / data_norm.sum()
    
    # 计算熵值
    n = len(data)
    e = -1/np.log(n) * np.sum(p * np.log(p + 1e-10), axis=0)
    
    # 计算权重
    w = (1 - e) / (1 - e).sum()
    return w
```
