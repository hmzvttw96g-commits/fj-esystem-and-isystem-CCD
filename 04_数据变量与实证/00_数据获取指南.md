
# 金融-教育-产业三维耦合研究数据获取指南

## 福建省数据（需手动获取）

### 1. 金融子系统数据
| 指标 | 数据来源 | 获取路径 |
|------|---------|---------|
| 金融业增加值/GDP | 福建省统计年鉴 | 国民经济核算 → 地区生产总值 |
| 金融深化度 | 人行福州支行 | 金融运行报告 |
| VC/PE对AI产业投资 | 清科研究中心 | pedata.cn |
| 政府AI基金 | 福建省发改委 | 官方公告 |

### 2. 教育子系统数据
| 指标 | 数据来源 | 获取路径 |
|------|---------|---------|
| AI相关专业招生占比 | 福建省教育厅 | 年度统计公报 |
| AI领域高层次人才数 | 教育部统计 | 学位中心统计 |
| AI领域论文+专利产出 | CNKI/国家专利局 | 官方数据库 |
| 校企联合实验室数 | 福建省科技厅 | 科技统计 |

### 3. 产业子系统数据
| 指标 | 数据来源 | 获取路径 |
|------|---------|---------|
| AI产业增加值 | 中国AI产业发展报告 | 信通院/艾瑞咨询 |
| AI企业数量 | 天眼查/企查查 | 企业数据库 |
| R&D投入强度 | 福建省科技厅 | 科技统计 |
| 技术合同成交额 | 福建省技术市场 | 技术市场统计 |

## 全国平均数据（需手动获取）

### 1. 全国金融数据
| 指标 | 数据来源 | 获取路径 |
|------|---------|---------|
| 全国金融业增加值/GDP | 中国统计年鉴 | 国民经济核算 |
| 全国金融深化度 | 人行报告 | 金融稳定报告 |

### 2. 全国教育数据
| 指标 | 数据来源 | 获取路径 |
|------|---------|---------|
| 全国AI招生占比 | 教育部统计 | 全国教育事业发展统计公报 |
| 全国AI高层次人才 | 教育部科技司 | 科技统计 |

### 3. 全国产业数据
| 指标 | 数据来源 | 获取路径 |
|------|---------|---------|
| 全国AI产业增加值 | 信通院报告 | AI产业发展报告 |
| 全国AI企业数量 | 信通院报告 | AI产业发展指数 |
| 全国R&D投入强度 | 科技部报告 | 全国科技经费投入统计公报 |

## 数据获取建议

1. **优先访问官方网站**：国家统计局data.stats.gov.cn
2. **联系相关部门**：福建省统计局、教育厅、科技厅
3. **使用商业数据库**：Wind、CNKI、企查查
4. **参考已有研究**：查阅已发表的论文中的数据引用

## 注意事项

1. AI产业统计数据在不同来源间可能存在口径差异
2. 建议使用多个数据源交叉验证
3. 对于估算数据，请在论文中明确标注数据来源
